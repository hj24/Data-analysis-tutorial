# 一个关于数据分析的简单易懂的教程

1. 教程的大概内容

- 包含常用的python机器学习库Scikit-Learn，Statsmodels，keras等模块的学习
- 对机器学习数据挖掘经典的重要的算法及问题进行复现，包含代码及数据集
- 参考数目：Python数据分析与挖掘实战，改正了原书的部分过时代码

- 本教程初衷是为了和队友一起准备建模比赛

- 还有没想到的以后补上
